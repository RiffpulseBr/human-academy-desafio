# RepurposeAI 🚀 | Desafio Human Academy

**RepurposeAI** é uma ferramenta de produtividade desenvolvida para agências de marketing e times de conteúdo. O objetivo é eliminar a fricção manual na adaptação de conteúdos entre diferentes canais digitais.

---

## 💡 O Problema (A Dor)
Copywriters e Analistas de Social Media perdem horas semanais formatando manualmente um mesmo conteúdo base para diferentes redes. Esse processo gera:
- **Ruído de comunicação:** Perda do tom de voz da marca.
- **Inconsistência:** Textos fora do limite de caracteres ou sem a estrutura ideal de cada rede.
- **Baixa Escalabilidade:** O time gasta tempo com formatação em vez de estratégia.

## ✨ A Solução
O RepurposeAI utiliza o motor **Gemini 2.5 Flash** para transformar um texto base (blog, roteiro ou anotações) em posts otimizados para Instagram, Threads (X) e LinkedIn com apenas um clique.

### Diferenciais Estratégicos:
- **Fricção Zero:** Sem necessidade de login ou autenticação, focado na agilidade do fluxo de trabalho.
- **Inteligência de Rede:** Prompts customizados para respeitar as nuances de cada plataforma (emojis, hashtags, tom de voz e formatação).
- **Segurança Pro:** Arquitetura com backend em Node.js protegendo chaves de API sensíveis via variáveis de ambiente.

## 🛠️ Stack Técnica
- **Frontend:** HTML5, CSS3 (Glassmorphism UI) e JavaScript Vanilla.
- **Backend:** Node.js + Express.
- **IA:** Google Generative AI (Gemini 2.5 Flash).
- **Infraestrutura:** Railway Pro (Deployment contínuo e escalável).
